# CORSAIR - Центрування тексту на головній сторінці

## Що змінено

### ✅ Текст на головній сторінці тепер по центру

**Зміни:**
- Весь текст hero секції тепер вирівняний по центру, як і на інших сторінках сайту
- Заголовок "Інноваційні рішення охолодження від CORSAIR" - по центру
- Підзаголовок "Ми створюємо високопродуктивні системи..." - по центру
- Кнопки "Переглянути продукцію" та "Зв'язатися з нами" - по центру
- Статистика (14+ років, 500K+ клієнтів, 99% відгуків) - по центру
- Анімація кулера розміщена під текстом, також по центру

**Технічні зміни в CSS:**

1. `.hero`:
   - Додано `flex-direction: column` (замість row)
   - Додано `text-align: center`
   - Змінено `justify-content` на `center`

2. `.hero-content`:
   - Змінено `max-width` з 600px на 900px для кращого читання
   - Додано `text-align: center`
   - Додано `width: 100%`

3. `.hero-buttons`:
   - Додано `justify-content: center` для центрування кнопок

4. `.hero-stats`:
   - Додано `justify-content: center` для центрування статистики

5. `.hero-animation`:
   - Видалено `flex: 1`
   - Додано `max-width: 600px` для обмеження ширини анімації

## Адаптивність

- **Десктоп (>1024px)**: Все по центру з анімацією під текстом
- **Планшет (768-1024px)**: Вже було по центру, залишилось так само
- **Мобільний (<768px)**: Все по центру, анімація зменшена

## Як використовувати

1. Розпакуйте `corsair-fixed-final.zip`
2. Відкрийте `index.html` в браузері
3. Перевірте головну сторінку - весь текст тепер по центру!
